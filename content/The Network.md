Fungal Network that connects all plants around the globe. Dense magic has the ability to transport individuals if convinced or if it believes that it is necessary to do so. Was Discovered by [[Clay the Red]] but no one would believe him and thought the old Druid was crazy. Many believe that Nature itself is connected but not physically, only magically. The network is a physical fungus network that can use magics. It typically does not as using magic off sets its balance in nature killing trees and other things in order to siphon magic to cast spells. the amount of vegetation destroyed equals the power of the spell. This network has never intervened before.
